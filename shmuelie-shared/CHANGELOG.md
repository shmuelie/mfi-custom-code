## [1.0.0] - 2026-03-29

### Added

- Initial release.
- String splitting with delimiter, preserving empty segments.
- String joining helpers with separator and optional concluder.
- Non-throwing integer parsing helpers: `try_stoul` and `try_stoi`.
- Numeric string validation with `is_number()`.
