# Shmuelie Shared

String helper functions (split, join, number parsing) shared across projects.

## Overview

A small utility library providing common string operations used by the mfi executables and libraries. All functions are in the `shmuelie` namespace.

## Dependencies

- No external dependencies.

## Details

- **Language**: C++20
- **Namespace**: `shmuelie`
- **Version**: 1.0.0
